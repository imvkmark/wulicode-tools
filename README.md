# Wulicode - App

Vite2.x + Vue3.x + TypeScript Backend Manager

## 快速开始

**安装依赖**

```bash
# 安装
$ yarn

# 启动
$ yarn start:<env>

# 打包
$ yarn build:<env>
```

**Sentry 不在开发环境下报告**

```
# config/.env.local

# 本地开发环境是 Development
NODE_ENV=development
```
